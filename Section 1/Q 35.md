## Factory 함수 사용하기

### 문제

> 2제곱, 3제곱, 4제곱을 할 수 있는 Factory 함수를 만들려고 합니다.<br> `<pass>`에 코드를 작성하여 two함수를 완성하세요.

```js
function one(n) {
    function two() {
        //pass
    }
    return two;
}

const a = one(2);
const b = one(3);
const c = one(4);

console.log(a(10));
console.log(b(10));
console.log(c(10));
```

### 코드 구현

```js
function one(n) {
    function two(num) {
        return num ** n;
    }
    return two;
}

const a = one(2);
const b = one(3);
const c = one(4);

console.log(a(10));
console.log(b(10));
console.log(c(10));
```

### 풀이과정

-   num

    -   타입 : number
    -   저장 데이터 : one함수의 변수로 들어가는 값

### 모범답안

```js
function one(n) {
    function two(value) {
        const sq = Math.pow(value, n);
        return sq;
    }
    return two;
}

const a = one(2);
const b = one(3);
const c = one(4);

console.log(a(10));
console.log(b(10));
console.log(c(10));
```
